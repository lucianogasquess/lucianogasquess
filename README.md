## 👤 Sobre mim

Tenho 18 anos e estou construindo minha trajetória no desenvolvimento de software com foco em **Front-End** — onde a lógica encontra a experiência do usuário. Mais do que apenas escrever código, me interessa criar interfaces que funcionem bem e que as pessoas queiram usar.

Paralelamente, venho estudando **Python, SQL e os fundamentos de Data Science** porque acredito que o desenvolvedor do futuro precisa entender não só como apresentar os dados, mas também como interpretá-los.

Estou em fase de construção de portfólio, com projetos reais que documentam minha evolução.
